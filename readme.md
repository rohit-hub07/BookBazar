# BookBazar

BookBazar is a platform for discovering, buying books online.

### Hosted Link: https://book-bazar-frontend-9q7i.vercel.app/

## Features

- Browse a wide selection of books
- User authentication and profiles
- Add books for sale
- See reviews ,add reviews
- Order details 

## Technologies Used

- Node.js
- Express
- MongoDB
- React
